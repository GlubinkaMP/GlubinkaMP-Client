# Glubinka MP — ClientRuntime

Клиентский пакет для подключения к серверам Glubinka MP.

## Скачать
Открой раздел **Releases** и скачай последнюю версию `GlubinkaMP-ClientRuntime`.

## Установка
1. Скачай ZIP из Releases
2. Распакуй архив
3. Скопируй содержимое в корневую папку GTA V

## Требования
- GTA V 1.0.2189.0
- Script Hook V
- ScriptHookVDotNet
- .NET Framework 4.7.2

## Запуск
Запусти GTA V и открой меню Glubinka MP в игре.
